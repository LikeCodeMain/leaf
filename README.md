# Leaf UI - Headless UI 组件库

Leaf UI 是一个基于逻辑与渲染分离架构设计的现代化 Headless UI 组件库，支持多框架使用。它提供了完整的 UI 组件、功能组件和 AI 前端组件，帮助开发者快速构建高质量的用户界面。

## 📖 项目介绍

Leaf UI 采用创新的架构设计，将组件的核心逻辑与具体框架的渲染实现完全分离。这种设计使得组件库既可以在不同框架中使用，又保持了高度的灵活性和可定制性。无论你使用 Vue、React 还是原生 JavaScript，都可以轻松接入 Leaf UI 的组件能力。

组件库分为三个核心模块：UI 组件提供基础界面元素，功能组件提供复杂业务场景解决方案，AI 前端组件则专门为 AI 应用场景提供开箱即用的组件支持。

## 🎯 预览

Leaf UI 提供了丰富的组件生态，从基础的按钮、输入框，到复杂的表格、表单，再到 AI 场景的对话流、文档解析等组件，覆盖了现代 Web 应用开发的各种需求。所有组件都遵循统一的设计规范，支持主题定制和国际化，确保在不同场景下都能提供一致的用户体验。

## 📦 安装与引入

Leaf UI 支持多种安装方式，你可以根据项目需求选择合适的方式。通过包管理器安装后，只需要简单的引入语句即可开始使用。组件库采用模块化设计，支持按需引入，不会增加不必要的代码体积。

## 🚀 第一个示例

开始使用 Leaf UI 非常简单。安装完成后，你只需要在项目中引入需要的组件，然后像使用普通组件一样使用即可。组件提供了丰富的配置选项，但大多数情况下使用默认配置就能满足需求。你可以从最简单的按钮组件开始，逐步探索更多功能。

## 📚 基础用法

Leaf UI 的组件设计遵循直观易用的原则。每个组件都提供了清晰的属性接口，通过简单的配置就能实现复杂的功能。组件支持事件回调、状态管理、样式定制等常见需求，同时内置了无障碍访问支持，确保应用的可访问性。

## 🔷 TypeScript 支持

Leaf UI 完全使用 TypeScript 开发，提供了完整的类型定义。在使用组件时，IDE 会提供智能提示和类型检查，帮助你减少错误并提高开发效率。所有组件的属性、事件、返回值都有详细的类型定义，确保类型安全。

## 🎨 主题定制入门

Leaf UI 支持灵活的主题定制能力。你可以通过主题配置来改变组件的颜色、尺寸、圆角等视觉属性，也可以完全自定义组件的样式。组件库提供了默认主题，同时也支持创建自定义主题，满足不同品牌和设计需求。

## 📋 组件列表

### UI 组件

**Button 按钮** - 提供多种样式的按钮组件，支持禁用、加载、图标等状态，适用于各种交互场景。

**Input 输入框** - 功能完整的输入框组件，支持多种输入类型、验证、清除等功能，提供良好的用户体验。

**Select 下拉选择** - 强大的下拉选择组件，支持单选、多选、搜索、分组等高级功能。

**Tabs 标签页** - 用于内容分组的标签页组件，支持多种样式和交互方式。

**Loading 加载指示器** - 提供多种样式的加载动画，用于表示异步操作进行中。

**Notification / Toast 通知** - 用于向用户展示提示信息的通知组件，支持多种类型和位置。

### 功能组件

**Form 表单** - 完整的表单解决方案，支持字段验证、动态表单、表单联动等复杂场景。

**Table 表格** - 功能强大的数据表格组件，支持排序、筛选、分页、选择等高级功能。

**Modal / Dialog 弹窗** - 灵活的弹窗组件，支持多种尺寸、位置和交互方式。

**Upload 文件上传** - 完整的文件上传组件，支持拖拽上传、进度显示、文件预览等功能。

**Export 数据导出** - 便捷的数据导出组件，支持多种格式和自定义导出逻辑。

**AuthGuard 权限控制** - 用于权限验证的组件，可以根据权限控制内容的显示和功能的可用性。

**ErrorBoundary 错误边界** - 用于捕获和处理组件错误的边界组件，提供优雅的错误处理机制。

**VirtualList 虚拟滚动** - 高性能的虚拟滚动列表组件，适用于大数据量场景。

**ThemeProvider 主题切换** - 主题提供者组件，支持动态切换主题和主题持久化。

**I18nWrapper 国际化** - 国际化包装组件，简化多语言应用的开发。

**DebounceInput 防抖节流** - 带有防抖或节流功能的输入组件，优化用户输入体验。

**SmartForm 智能表单** - 基于配置的智能表单组件，可以根据数据自动生成表单结构。

### AI 前端组件

**ChatStream 流式对话** - 专门为 AI 对话场景设计的组件，支持流式消息展示和丰富的交互功能。

**DocParser 文档解析器** - 用于解析和展示文档内容的组件，支持多种文档格式。

**AIBlock AI编辑器块** - 用于 AI 编辑器的内容块组件，支持多种内容类型和编辑操作。

**ToolCallPanel 工具调用面板** - 展示 AI 工具调用过程的组件，帮助用户理解 AI 的执行流程。

**PromptDebugger 提示词调试器** - 用于调试和优化 AI 提示词的组件，提供可视化的调试界面。

**RAGResultList 向量检索结果** - 展示 RAG 检索结果的组件，支持结果高亮和交互操作。

**AgentStatusBoard Agent状态看板** - 用于展示 AI Agent 运行状态的看板组件，提供实时状态监控。

## 🏗️ 模块介绍

### 快速开始

快速开始模块提供了项目的基本介绍、安装指南和第一个示例。无论你是第一次接触 Leaf UI，还是想要快速了解组件库的能力，都可以从这里开始。这里会引导你完成环境准备、依赖安装和第一个组件的使用，让你快速上手。

### UI 组件

UI 组件模块包含了所有基础的用户界面元素。这些组件是构建应用界面的基础，提供了按钮、输入框、选择器、标签页等常见 UI 元素。每个组件都经过精心设计，确保在不同场景下都能提供良好的用户体验。组件支持丰富的配置选项，同时保持了简单易用的特性。

### 功能组件

功能组件模块提供了解决复杂业务场景的组件集合。这些组件不仅仅是简单的 UI 元素，而是包含了完整业务逻辑的解决方案。从表单处理到数据展示，从文件上传到权限控制，这些组件帮助开发者快速实现复杂的业务需求，减少重复开发工作。

### AI 前端组件

AI 前端组件模块专门为 AI 应用场景设计，提供了从对话交互到工具调用，从文档解析到状态监控的完整组件生态。这些组件充分考虑了 AI 应用的特殊需求，如流式数据展示、实时状态更新、复杂交互流程等，帮助开发者快速构建现代化的 AI 应用界面。
