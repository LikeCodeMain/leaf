# UI 组件模块

UI 组件模块包含基础的用户界面组件，这些组件提供基本的交互功能。

## 📦 组件列表

### Button（按钮）
- 位置：`ui/button/`
- 状态：✅ 已实现
- 说明：提供按钮的基础逻辑和 Vue 实现

### Input（输入框）
- 位置：`ui/input/`
- 状态：🚧 待实现

### Select（选择器）
- 位置：`ui/select/`
- 状态：🚧 待实现

## 🎯 设计原则

- **逻辑与渲染分离**：每个组件都有 `core/` 目录存放框架无关的逻辑
- **多框架支持**：通过 `vue/`、`react/` 等目录实现不同框架的渲染层
- **类型安全**：完整的 TypeScript 类型定义

## 📝 添加新 UI 组件

1. 在 `ui/` 目录下创建组件目录
2. 创建 `core/` 目录存放核心逻辑
3. 创建 `vue/` 目录实现 Vue 组件
4. 在 `ui/index.ts` 中导出
